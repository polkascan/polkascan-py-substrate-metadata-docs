accept_channel_transfer¶

Accepts channel transfer. commitment_params is required to prevent changing the transfer conditions.

<weight>

## Weight O (A) where: - A is the number of entries in commitment_params.new_collaborators map - DB: - O(1) doesn&#x27;t depend on the state or parameters # </weight>

Attributes¶

Name Type
channel_id T::ChannelId
commitment_params TransferCommitmentWitnessOf<T>

Python¶

call = substrate.compose_call(
    'Content', 'accept_channel_transfer', {
    'channel_id': 'u64',
    'commitment_params': {
        'new_collaborators': 'scale_info::140',
        'price': 'u128',
        'transfer_id': 'u64',
    },
}
)
