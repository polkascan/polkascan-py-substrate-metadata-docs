reveal_vote¶

Reveal a sealed vote in the referendum.

# <weight>

## Weight O (W) where: - W is the number of intermediate_winners stored in the current Stage::&lt;T, I&gt;::get() - DB: - O(1) doesn&#x27;t depend on the state or parameters # </weight>

Attributes¶

Name Type
salt Vec<u8>
vote_option_id <T as common::membership::MembershipTypes>::MemberId

Python¶

    'Referendum', 'reveal_vote', {
    'salt': 'Bytes',
    'vote_option_id': 'u64',
}
)

Storage functions¶

AccountsOptedOut¶

Accounts that permanently opted out of voting in referendum.

Python¶

result = substrate.query(
    'Referendum', 'AccountsOptedOut', ['AccountId']
)

Return value¶

()

Stage¶

Current referendum stage.

Python¶

result = substrate.query(
    'Referendum', 'Stage', []
)

Return value¶

{
    'Inactive': None,
    'Revealing': {
        'current_cycle_id': 'u64',
        'ends_at': 'u32',
        'intermediate_winners': [{'option_id': 'u64', 'vote_power': 'u128'}],
        'started': 'u32',
        'winning_target_count': 'u32',
    },
    'Voting': {
        'current_cycle_id': 'u64',
        'ends_at': 'u32',
        'started': 'u32',
        'winning_target_count': 'u32',
    },
}

Votes¶

Votes cast in the referendum. A new record is added to this map when a user casts a sealed vote. It is modified when a user reveals the vote's commitment proof. A record is finally removed when the user unstakes, which can happen during a voting stage or after the current cycle ends. A stake for a vote can be reused in future referendum cycles.

Python¶

result = substrate.query(
    'Referendum', 'Votes', ['AccountId']
)

Return value¶

{
    'commitment': 'scale_info::11',
    'cycle_id': 'u64',
    'stake': 'u128',
    'vote_for': (None, 'u64'),
}
