unstake¶

Unstake by exchange derivative for assets, the assets will not be available immediately. Instead, the request is recorded and pending for the nomination accounts on relaychain chain to do the unbond operation.

  • amount: the amount of derivative

Name Type

    'PalletLiquidStaking', 'unstake', {
    'liquid_amount': 'u128',
    'unstake_provider': (
        'RelayChain',
        'Loans',
        'MatchingPool',
    ),
}
