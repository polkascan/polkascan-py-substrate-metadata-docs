### LastAssetId
 The last processed asset id - used when assigning a sequential id.

#### Python
```python
result = substrate.query(
    'AssetRegistry', 'LastAssetId', []
)
```

#### Return value
```python
{
    'Native': None,
    'Stellar': {
        'AlphaNum12': {'code': '[u8; 12]', 'issuer': '[u8; 32]'},

        'StellarNative': None,
    },
    'XCM': 'u8',
    'ZenlinkLPToken': ('u8', 'u8', 'u8', 'u8'),
}
```
