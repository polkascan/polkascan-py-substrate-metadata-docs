
# OraclePriceFeed

---------
## Calls

---------
### feed
Permissionles call to feed an oracle key from a source with value.
The first time a value is set for a key, an extra fee is required
for the feeder.
#### Attributes
| Name | Type |
| -------- | -------- | 
| key | `T::OracleKey` | 
| value | `T::OracleValue` | 

#### Python
```python
call = substrate.compose_call(
    'OraclePriceFeed', 'feed', {
    'key': {
        'ConversionRatio': (
            {
                'Native': None,
                'Tranche': (
                    'u64',
                    '[u8; 16]',
                ),
                None: None,
                'AUSD': None,
                'ForeignAsset': 'u32',
                'LocalAsset': 'u32',
                'Staking': (
                    'BlockRewards',
                ),
            },
            {
                'Native': None,
                'Tranche': (
                    'u64',
                    '[u8; 16]',
                ),
                None: None,
                'AUSD': None,
                'ForeignAsset': 'u32',
                'LocalAsset': 'u32',
                'Staking': (
                    'BlockRewards',
```
