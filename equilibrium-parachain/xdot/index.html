Calls¶

burn¶

Burn liquidity tokens in exchange for base and fyToken or base only with trade_to_base=true

Attributes¶

Name Type
pool_id PoolId
min_ratio (u32, u32)
max_ratio (u32, u32)
tokens_burned T::Balance
trade_to_base bool

Python¶

call = substrate.compose_call(
    'Xdot', 'burn', {
    'max_ratio': ('u32', 'u32'),
    'min_ratio': ('u32', 'u32'),
    'pool_id': 'u32',
    'tokens_burned': 'u128',
    'trade_to_base': 'bool',
}
)

create_pool¶

Creates pool - maturity: unix timestamp when maturity is coming - ts_period: period in secs for ts coeff

    },

        'bits': 'i128',

}
)

mint¶

Mint liquidity tokens, with an optional internal trade to buy xbase tokens beforehand. The amount of liquidity tokens is calculated from the amount of xbase tokens to buy from the pool, plus the xbase_in. A proportional amount of base tokens need to be sent. It fails if amount of base tokens for trade less than base_in

Attributes¶

Name Type
pool_id PoolId
min_ratio (u32, u32)
max_ratio (u32, u32)
base_in T::Balance
xbase_in T::Balance
xbase_to_buy T::Balance

Python¶

call = substrate.compose_call(
    'Xdot', 'mint', {
    'base_in': 'u128',
    'max_ratio': ('u32', 'u32'),
    'min_ratio': ('u32', 'u32'),
    'pool_id': 'u32',
    'xbase_in': 'u128',
    'xbase_to_buy': 'u128',
}
)
