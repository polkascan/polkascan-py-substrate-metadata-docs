Currencies¶

Calls¶

burn¶

Users destroy their own assets.

Attributes¶

Name Type
currency_id AssetId
amount BalanceOf<T>

Python¶

call = substrate.compose_call(
    'Currencies', 'burn', {
    'amount': 'u128',
    'currency_id': 'u32',
}
)

create_asset¶

Users create the asset.

Attributes¶

Name Type
currency_id AssetId
amount BalanceOf<T>
metadata Option<DicoAssetMetadata>

Python¶

call = substrate.compose_call(
    'Currencies', 'create_asset', {
    'amount': 'u128',
    'currency_id': 'u32',
    'metadata': (
        None,
        {
            'decimals': 'u8',
            'name': 'Bytes',
            'symbol': 'Bytes',
        },
    ),
}
)
